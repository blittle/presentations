Installing NodeJS

http://nodejs.org/
Windows - Download .msi installer
MacOS - Download .pkg installer
Linux - Download source and build

Building on Linux

  1. sudo apt-get install build-essential python libssl-dev
  2. tar xvf node-v0.10.1.tar.gz
  3. cd node-v0.10.1
  4. ./configure
  5. make
  6. sudo make install
